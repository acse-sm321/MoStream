## Access the website
[MoStream](https://mostream.com)

## Run & installation
```
$ git clone
$ cd
$ go install
$ cd
$ ./main

# open in browser http://localhost:9999/
```
Prerequisites:  Golang and dependencies in go.mod pre-installed.
